Font-family

Default value

Sets a specific font for the text.
Syntax
font-family: "font (family) name"

Hello World

Internal Spesific Font {font-family: Arial, Helvetica, sans-serif}

  • Browser tries Arial first.
  • If it's not possible to use then tries to Halvetica.
  • If it's not possible too, then tries sans-serif.
  • If last one also not possible to use, then browser uses default font.

Hello World

External Spesific Font Via HTML link {font-family: 'Sansita Swashed', cursive}

Externally fonts can be used in HTML via <link href>

Hello World

External Spesific Font Via CSS Import {font-family: 'Josefin Sans', sans-serif}

Externally fonts can be used in HTML via @import url()

Font-size

Default value

defines the color of the text.
Syntax
font-size: "size + unit"

Hello World

Relative size {font-size: 1.2em;}




Calculates the font sizes about parent font size. Parent Font Size * XYZem
Parent container: 20px;
font-size: 1.2em; (24px)
font-size: 1.2em; (28.8px)
font-size: 1.2em; (34.56px)
